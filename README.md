
# Fortran (95)

Fortran is a general purpouse compiled imperative programming language. This template on Replit allows you to write, compile and run Fortran 95 code.


## Usage/Examples

Print Hello, World!
```fortran
program hello
  print *, 'Hello, World!'
end program hello
```
Variable declaration
```fortran
program var
  integer :: year
  real :: money
  complex :: root
  character :: letter
  logical :: on
end program var
```
